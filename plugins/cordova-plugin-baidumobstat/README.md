# 百度移动统计Cordova插件

## How to debug:
1. clone this project
2. remove the old plugin from the project
  ```
  cordova plugin remove cordova-plugin-baidumobstat
  ```

3. add the plugin

  	```
  	cordova plugin add cordova-plugin-baidumobstat
  	```

4. config
	
	参考cordova-sample部分

4. build and debug
	
	```
	cordova build
	cordova run android  (cordova run ios)
	```
